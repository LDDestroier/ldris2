# LDRIS version 2
Modern multiplayer tetris for ComputerCraft. (Work in progress)

### Current features:
- SRS rotation and wall-kicking
- 7-bag randomization
- Modern-feeling controls
- Ghost piece
- Piece holding
- Sonic drop
- Configurable SDF, DAS, ARR, ARE, lock delay
- Animated piece queue
